WoWJapanizer Readme

■ WoWJapanizerについて

WoWJapanizerは、米Blizzard社のMMORPG "World of Warcraft" の一部を日本語化するaddonです。
現在、以下の日本語化に対応しています。

・Quest (QuestLog、WorldMap Quest)
　【機能】
　　- QuestLogの日本語訳の表示化。
　　- WorldMapのQuestデータの日本語訳の表示。
　　- Quest受領前(Accept)にQuestの日本語訳の表示(一部非対応のものあり)。
　　- Quest完了時(Complate)にQuestの日本語訳の表示(一部非対応のものあり)。
　　- Quest ToolTip(チャットのリンクのポップアップ)に日本語訳を表示(一部非対応のものあり)。
　【翻訳データ】
　　- 機械翻訳(excite翻訳)
　　- 人による翻訳

・Item ToolTip
　【機能】
　　- Item ToolTipに日本語訳を表示。
　【翻訳データ】
　　- 人による翻訳

・Spell ToolTip
　【機能】
　　- Spell ToolTipに日本語訳を表示。
　【翻訳データ】
　　- 人による翻訳

・Achievement
　【機能】
　　- Achievementの日本語訳と攻略アドバイス。ToolTipで日本語訳と攻略アドバイスを表示します。
　【翻訳データ】
　　- 人による翻訳

人による日本語翻訳は、有志の手によって支えられています。
1つでも2つでも協力いただけると助かります。(後述)


■ WoWJapanizerの構成

WoWJapanizerは、プログラム本体と日本語翻訳データとに分けてあります。
プログラム本体には、フォントファイルが含まれています。
分けてあるのは、日本語翻訳データの更新を、ある程度定期的に行えるようにするためです。

【WoWJapanizer】
　プログラム本体と日本語フォントファイルです。
　WoWJapanizerがないと、日本語翻訳機能を使用することができません。

■ WoWJapanizerの設定について

World of Warcraftの設定
以下の2つの設定を行って下さい。（初期設定だとチェックが入っているはずです）

1. メニュー -> [Interface] -> [Help] -> Enhanced Tooltips にチェックを入れる
2. メニュー -> [Interface] -> [Display] -> Display Points as Average にチェックを入れる

WoWJapanizerの設定

メニュー -> [Interface] -> [addon]のタブ -> WoWJapanizer
